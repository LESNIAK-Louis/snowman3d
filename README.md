# snowman3d

Louis LESNIAK et Théo JOFFROY

## Compilation

```
git clone https://github.com/lesniak4/snowman3d.git &&
cd snowman3d &&
mkdir build &&
cd build &&
cmake .. &&
make
```
## Exécution
```
./snowman3d
```
